Создание .Net модуля Ингео 


1. Создать проект "Class Library"
2. В список References добавить COM библиотеки ("ИнГео - вспомогательные объекты", "ИнГео - ГИС")  
  (меню Project\Add Reference..., закладка COM)
3. В класс обязательно добавить метод инициализации (связывания модуля с ИнГео), одним из параметров которого должна быть ссылка на IIngeoApplication
4. При необходимости добавить метод деинициализации
5. Реализовать требуеюмую функциональность
6. В свойствах .Net проекта установите аттрибут [ComVisible(true)], либо установите указанный аттрибут у классов которые должны быть видны из COM-а 
7. Зарегистрировать .Net модуль как COM объект 
  regasm модуль /codebase
8. Создать в ИнГео скрипт для подключения созданной библиотеки к ИнГео и вызова необходимых функций

9. Для запуска модуля запустите скрипт созданный в пункте 8
